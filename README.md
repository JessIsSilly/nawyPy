# nawyPy
A python port of the multipurpose discord bot built in rust Nawy

# Why did i port this you may ask?
Got bored

thats kinda it

# Ported Feature list:
- [ ] ping (no bot will be complete without a ping command)
- [ ] coinflip (we love flipping coins to decide on things)
- [ ] time (tells time, what'd you expect :p)
- [ ] kick (get em out)
- [ ] ban (get em out permanently)
- [ ] meow (meows at you when you meow)
- [ ] cat (gives a random cat pic :3)
- [ ] help (lists commands(should have done earlier tbh))
- [ ] pingr (ping remote server)
- [ ] info (info about the bot)
# how to run
```
cargo run
```
its as shrimple as that :3c

# how to use it
the bot uses / commands instead of the . prefix of the original bot
